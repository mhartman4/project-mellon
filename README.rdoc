== Project Mellon Rails App!

The home page runs javascript-based linear regression on all games played by NBA players in '13-'14. You can also explore individual player performances over the course of last season.

Fanduel Scoring For Reference

* 3-pt FG = 3pts
* 2-pt FG = 2pts
* FT = 1pt
* Rebound = 1.2pts
* Assist = 1.5pts
* Block = 2pts
* Steal = 2pts
* Turnover = -1pt
